Make sure you’ve installed:

  • Git for Windows
  • Python 3 (during installation, check “Add Python to PATH”)
  • Microsoft C++ Build Tools (for compiling dependencies)

    • Installation Steps in Windows Terminal or PowerShell

    Step 1: Install required Python packages and build tools (Make sure pip is working after installing Python)

    pip install --upgrade pip setuptools wheel
    pip install httpx

    Step 2: Clone SearXNG repo git clone https://github.com/searxng/searxng.git cd searxng # Step 3: Create and activate a virtual environment python -m venv venv venv\Scripts\activate # Step 4: Install SearXNG dependencies pip install --use-pep517 --no-build-isolation -e . # Step 5: Launch the server python searx\webapp.py Accessing the Local Server Once it’s running, open your browser and visit: http://127.0.0.1:8888
