Creating a website with div containers involves structuring your HTML content and then styling those div elements with CSS to achieve the desired layout.

1. HTML Structure with div Elements:

  • Container div: Typically, you'll start with a main div container to wrap your entire website content, often used to center the content and set a maximum width.

    <div class="container">

        <!-- All your website content goes here -->

    </div>

  • Section divs: Within the main container, you can use div elements to define distinct sections of your page, such as a header, navigation, main content area, and footer.

    <div class="container">

        <div class="header">

            <!-- Header content like logo, title -->

        </div>

        <div class="navigation">

            <!-- Navigation links -->

        </div>

        <div class="main-content">

            <!-- Main content of the page -->

            <div class="left-column">

                <!-- Content for the left column -->

            </div>

            <div class="right-column">

                <!-- Content for the right column -->

            </div>

        </div>

        <div class="footer">

            <!-- Footer content like copyright info -->

        </div>

    </div>

  • Content divs: Further nest div elements within sections to organize specific content blocks (e.g., columns within a main content area, individual cards in a gallery).

2. CSS Styling for div Containers:

  • Basic Styling: Use CSS to define the appearance of your div elements. You can target them by their class or id attributes.

    .container {

        width: 960px; /* Example fixed width */

        margin: 0 auto; /* Center the container */

        background-color: #f0f0f0;

        padding: 20px;

    }

    .header {

        background-color: #333;

        color: white;

        padding: 10px;

    }

    .navigation ul {

        list-style: none;

        padding: 0;

    }

    .navigation li {

        display: inline-block;

        margin-right: 15px;

    }

  • Layout Techniques:
  • Floats: Use float: left; or float: right; on div elements to position them side-by-side (e.g., for columns). Remember to use a "clearfix" or overflow: hidden; on the parent container to prevent layout issues with floated elements.
  • Flexbox: A more modern and powerful way to arrange items in a one-dimensional layout (row or column). Apply display: flex; to the parent div and use properties like justify-content and align-items to control the children's arrangement.
  • CSS Grid: For two-dimensional layouts, CSS Grid offers advanced control over rows and columns. Apply display: grid; to the parent div and define grid properties like grid-template-columns and grid-template-rows.

Example using Flexbox for columns:

<div class="main-content-flex">

    <div class="flex-column">Column 1</div>

    <div class="flex-column">Column 2</div>

</div>

.main-content-flex {

    display: flex;

    justify-content: space-between; /* Distribute space between columns */

}

.flex-column {

    flex: 1; /* Allows columns to grow and shrink */

    padding: 15px;

    background-color: lightblue;

    margin: 0 10px;

}
