Ringzauber

Ringzauber is a private browser made by Python that does not save your history. It is easy use by anyone and aviable to use on Windows 7, 8.1 and 10 and 11.

Ringzauber Logo

Best used with SearXNG(note, we did not create this)! Download SearXNG in your terminal:

Copy this line: sudo apt update
sudo apt install -y git python3 python3-venv python3-pip build-essential libxslt-dev zlib1g-dev libffi-dev libssl-dev


Paste this in next:git clone https://github.com/searxng/searxng.git
cd searxng


Next:python3 -m venv venv
source venv/bin/activate


Next:pip install --upgrade pip setuptools wheel
pip install --use-pep517 --no-build-isolation -e .


Finally: enter this:python searx/webapp.py


Open this url in Ringzauber:http://127.0.0.1:8888
